# GST Bill Generator

Simple static web app for calculating:

- taxable value before GST
- total GST amount
- CGST 9% + SGST 9% split
- or IGST 18%

## Run locally

Open `index.html` in a browser.

## Deploy on Vercel

1. Push the `bill-generator` folder to GitHub.
2. Import the project into Vercel.
3. Set the root directory to `bill-generator` if needed.
4. Deploy with the default static settings.
